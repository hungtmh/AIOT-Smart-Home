# AIOT Smart Home

Project layout:

```text
backend/   Spring Boot API, Supabase PostgreSQL, MQTT bridge
frontend/  React + Vite dashboard
arduino/   Physical board sketches
wokwi/     Wokwi simulator projects
```

Run backend:

```powershell
cd backend
.\run-dev.ps1
```

Run frontend:

```powershell
cd frontend
npm run dev
```
